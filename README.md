## Getting Started
### Run locally
1. Clone this repo
  ```
  git clone 
  ```
2. Navigate to local directory
3. Install dependencies
  ```
  npm install 
  ```
4. Run the client 
  ```
  npm run dev 
  ```
5. Setup your MetaMask browser wallet - see https://metamask.io/download.html to download.
